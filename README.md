# 📊 Business Sales Analytics Dashboard

A data analytics project analysing 9,994 sales records from a retail superstore to extract business insights and visualise KPIs.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL
- Jupyter Notebook

## 📈 Key Insights Uncovered
- Technology is the top revenue category at $836,154 with a 17.4% profit margin
- Furniture has dangerously low profit margins at only 2.49% despite high sales
- The West region is the most profitable region with a 14.94% profit margin
- Sales peak consistently in September and November every year

## 📁 Project Files
- `sales_analysis.ipynb` — Full analysis notebook
- `sales_dashboard.png` — 4-panel visual dashboard
- `sales_report.xlsx` — Automated Excel summary report

## 🚀 How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas matplotlib seaborn openpyxl`
3. Open `sales_analysis.ipynb` in Jupyter Notebook
4. Run all cells
